# MountainsMeter

## Precondition

* Mac OS X Yosemite
* node.js (v5.4.0 ~)
* npm (3.3.12 ~)

## Getting started

### 1. Download this repository

```bash
$ git clone {this repository's URL}

$ cd mountainsmeter
```

### 2. Install the Node packages as Glocal

```bash
$ npm install -g gulp-cli
```

### 3. Install the Node packages as Local

```bash
$ npm install
```

### 4. Run Task

```bash
$ gulp
```
